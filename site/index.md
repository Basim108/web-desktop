# Bookmark Desktop

A Chrome extension that replaces the new-tab page with a desktop of your
bookmarks — a customizable grid of icons with a folder sidebar.

- [Privacy policy](privacy/) — what the extension does with your data, and
  where it stays.
- [Report a bug or request a feature](https://github.com/Basim108/bookmark-desktop/issues)
  — found something broken, or want the extension to do something it doesn't?
  Open an issue here. A GitHub account is required.
- [Source code on GitHub](https://github.com/Basim108/bookmark-desktop) — the
  full source, MIT licensed.
